# 🪣 Enigma dos Baldes (L2.E1) - Jogo Interativo

Este projeto é uma ferramenta pedagógica interativa desenvolvida em HTML5, CSS3 e JavaScript. O objetivo é ajudar alunos a resolverem o clássico enigma de lógica: **Como obter exatamente 4 litros de água usando apenas dois baldes de 3L e 5L?**

## 🎯 O Desafio

Tens à tua disposição:
1. Um balde com capacidade de **3 Litros**.
2. Um balde com capacidade de **5 Litros**.
3. Um reservatório de água ilimitada.

**Regras:**
* Podes encher um balde totalmente.
* Podes esvaziar um balde totalmente.
* Podes transferir água de um balde para o outro até que o destino esteja cheio ou a origem vazia.
* **Objetivo Final:** Deixar exatamente **4 Litros** num dos baldes.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura da página.
* **CSS3:** Design moderno em "Dark Mode" e animações de feedback.
* **Canvas API:** Utilizada para renderizar a simulação visual da água dentro dos baldes de forma dinâmica.
* **JavaScript (Vanilla):** Lógica matemática para as transferências e verificação de vitória.

## 💻 Como Utilizar

1. Faz o download do ficheiro `index.html`.
2. Abre o ficheiro em qualquer navegador web moderno (Chrome, Firefox, Edge, Safari).
3. Utiliza os botões de controlo para manipular os volumes:
   - **Encher:** Enche o balde selecionado até à capacidade máxima.
   - **Esvaziar:** Remove toda a água do balde.
   - **Passar:** Transfere o conteúdo respeitando o limite do balde de destino.

## 📖 Contexto Pedagógico

Este exercício é excelente para ensinar:
* **Pensamento Computacional:** Decomposição de problemas e algoritmos.
* **Aritmética Modular:** A relação entre os volumes e o máximo divisor comum.
* **Resolução de Problemas:** Estratégias de tentativa e erro guiada.

---
**Nota:** Este projeto foi desenhado para ser projetado em sala de aula, permitindo que o professor medie a atividade enquanto os alunos sugerem os passos logicamente.
